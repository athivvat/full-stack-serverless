# Full-Stack Serverless on AWS
Building a Full-Stack Serverless Application on AWS

## Requirements
```shell
# NPM
npm install -g npm

# AWS CLI
npm install -g aws-cli

# AWS CDK CLI
npm install -g aws-cdk

# TypeScript (Optional: If not using TypeScript)
npm install -g tsc
```
